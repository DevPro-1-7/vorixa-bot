# ⚡ CharglyBot V3 — diigiure project

> **Pre-Approval Digital Commerce System**
> منصة شحن ألعاب احترافية داخل تيليجرام

---

## 🔄 تدفق الطلب الكامل

```
🎮 اختيار اللعبة
  ↓
📝 Player ID + اسم اللاعب + صورة البروفايل
  ↓
📦 اختيار الباقة
  ↓
📨 [إشعار الأدمن: صورة + بيانات + أزرار]
  ↓
┌─ ✅ قبول الطلب
│     ↓
│   💳 تفاصيل الدفع (مهلة 15 دقيقة)
│     ↓
│   📸 المستخدم يرسل صورة إثبات الدفع
│     ↓                    🔒 Protected Order
│   [الأدمن يراجع الإثبات]
│     ↓
│  ┌─ ✅ تم الشحن  →  🎉 إشعار للمستخدم
│  ├─ ⚡ قيد التنفيذ → إشعار + تحديث لاحق
│  └─ ⚠️ مشكلة تحويل → إشعار مع ملاحظة
│
└─ ❌ رفض الطلب → إشعار مع السبب
```

---

## 📊 State Machine

| الحالة | المعنى | قابل للتعديل؟ | محمي؟ |
|--------|--------|--------------|-------|
| `awaiting_approval` | ينتظر مراجعة الأدمن | ✅ نعم | ❌ |
| `waiting_payment` | وافق الأدمن — ينتظر الدفع | ❌ لا | ❌ |
| `payment_sent` | إثبات وصل | ❌ لا | 🔒 نعم |
| `processing` | قيد التنفيذ | ❌ لا | 🔒 نعم |
| `completed` | ✅ تم الشحن | ❌ | — |
| `rejected` | ❌ مرفوض | ❌ | — |
| `expired` | ⏰ انتهت المهلة | ❌ | — |
| `cancelled` | 🚫 ألغاه المستخدم | ❌ | — |

## 👑 لوحة تحكم الأدمن

أرسل `/admin` — تظهر لوحة تحكم كاملة:

### إدارة الطلبات
| الزر | الوظيفة |
|------|---------|
| 🔍 مراجعة | الطلبات الجديدة بانتظار القبول/الرفض |
| 💳 إثباتات دفع | Protected Orders بانتظار تأكيد الشحن |
| ⚡ قيد التنفيذ | طلبات تحت التنفيذ |
| 📊 إحصائيات | أرباح + صافي ربح + أكثر الألعاب طلباً |

### إدارة الألعاب والباقات
| الإجراء | الطريقة |
|---------|---------|
| إضافة لعبة جديدة | من لوحة التحكم ← إدارة الألعاب ← ➕ |
| تفعيل/إيقاف لعبة | زر 🔄 بجانب اللعبة |
| حذف لعبة | زر 🗑️ (يحذف باقاتها أيضاً) |
| إضافة باقة | من باقات اللعبة ← ➕ |
| تعديل سعر باقة | زر التعديل ← سعر جديد |
| تفعيل/إيقاف باقة | زر 🔄 |

### الإشعار الجماعي
من لوحة التحكم ← 📢 إشعار جماعي ← أرسل النص

## 🛡️ الأمان

| الميزة | التفصيل |
|--------|---------|
| لا دفع قبل الموافقة | ❌ مستحيل |
| طلب نشط واحد فقط | يمنع التلاعب |
| Protected Order | الأدمن لا يستطيع رفض الطلب بعد إثبات الدفع |
| تعديل مقفول بعد الموافقة | يمنع التحايل |
| Audit Log كامل | كل عملية مسجّلة بوقتها |
| مهلة 15 دقيقة | إلغاء تلقائي عند انتهائها |
| حماية السبام | 5 ثواني بين كل طلب |
